‘Edit Document Archive Settings’

 

This page allows you to modify archiving settings for a document:

 

  1. Archive By Date:
    1. Either enter an expiration date ie. when the document will be archived.
    2. Or specify a period from the document creation date ie. 3 Months after creation.
  2. Archive By Utilisation:
    1. Specify the document transaction that will trigger document archival.
    2. Enter the period after which the document will be archived.

Eg. If you select the ‘Download’ transaction and ‘3 days’ as the period, then the document will be archived 3 days after the last Download.

  1. The ‘Clear’ button resets the period and date information.
  2. The ‘Update’ button saves the current information.
  3. The ‘Remove’ button removes the current archiving settings.
  4. The ‘Cancel’ button returns you to the document details page.
